Create Personal Page on AFS

There are many ways to create personal page on server Web.njit.edu which is accessible to all njit students and faculties.

Demo

  1. My Personal Web Page

Prerequisite

  1. NJIT UCID account

  2. SSH Secure Shell

  3. Jemdoc installation (optional)

Method

  1. Open a ssh client and login to your AFS account 

ssh username@afsx.njit.edu

  1. Here

    1. Username is your UCID

    2. Replace x with 1 to 36

    3. Password is UCID password

For example 

ssh ts362@afs1.njit.edu

  1. Create a folder in your computer which will contain the your site 

mkdir ~/site-folder

  1. You can write your html web page. Java programs, PHP, JSP, and MySQL are currently supported. Or you can download light static markup langauge jemdoc.

  1. Push the completed site to your personal web space on http:www.princeton.edu to make the site public.

rsync -avze ssh  ~/site-folder/ username@afsx.njit.edu:public_html/
Note

The tutorial ends here unless you're using jemdoc

  1. Download the css file for jemdoc. 

curl -o ~/site-folder/jemdoc.css http://jemdoc.jaboc.net/dist/jemdoc.css

  1. Compile the jemdoc file 

pushd ~/site-folder 1>/dev/null ; jemdoc *.jemdoc ; popd 1>/dev/null

  1. push the site to the public server 

rsync -avze ssh --exclude '*.jemdoc' --exclude 'MENU' ~/site-folder/ username@afsx.njit.edu:public_html/
